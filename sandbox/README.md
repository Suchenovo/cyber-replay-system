# 沙箱环境说明

## 功能
- 提供隔离的网络环境用于安全的流量重放
- 防止重放流量影响生产网络
- 支持流量分析和捕获

## 网络隔离
- 使用Docker内部网络（internal network）
- 通过iptables规则进一步限制通信
- 仅允许沙箱网络内部通信（172.20.0.0/16）

## 工具
- tcpdump: 流量捕获
- tshark/wireshark: 流量分析
- scapy: Python流量操作库
- iptables: 网络规则管理

## 使用方法
沙箱容器会自动启动并配置网络隔离规则。
后端服务可以通过Docker API与沙箱容器交互，执行流量重放任务。
